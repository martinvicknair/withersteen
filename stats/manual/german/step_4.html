Schritt 4 - Javascript in Webseiten einbinden


Um zu gewährleisten, dass auch wirklich alle Besucher gezählt werden, die Ihre Webseite besuchen, müssen Sie folgenden JavaScript in jede aufzuzeichnende Webseite integrieren. Bitte kopieren Sie dazu das folgende Javascript in jede aufzuzeichnende Seite zwischen <body> und </body>:

<script src="http://www.example.com/stat/pws.php?mode=js"></script>

Handelt es sich bei Ihrer Webseite um XHTML, so richten Sie sich an folgende Vorlage:

<script type="text/javascript" src="http://www.example.com/stat/pws.php?mode=js"></script>

Bitte achten Sie darauf, dass der vollständige Pfad zu der auf dem Server liegenden PWS.PHP angegeben werden muss!
Da es sich hierbei um einen externen Pfad handelt, kann diese Zeile auch unverändert in Unterverzeichnissen des Server verwendet werden.

Einfügen in eine reine PHP Webseiten

Wir liefen Ihnen mit der Datei "config/tracking_code.php" eine Vorlage, den Code zur Aufzeichnung konfortabel per include einbinden zu können.

<?php include ( 'stat/config/tracking_code.php' ) ?>


Sollte Ihre Seite für XHTML ausgelegt sein, so nutzen Sie:

<?php include ( 'stat/config/tracking_code_xhtml.php' ) ?>

Nun sind Sie fertig und können über Ihren Browser die index.php aufrufen, um sich die Statistik anzusehen.
