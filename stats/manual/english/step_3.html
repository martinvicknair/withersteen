Step 3 - Setup


Now open your browser and type in the address of the setup script. Replace www.example.com with your own domain name:

http://www.example.com/stat/config/setup.php?lang=en

The default password is admin. You should change this immediately after logged in for the first time, under Advanced Settings.


If you want to use the setup in your favorite language, please use the following parameter right after setup.php in your browser navigation bar:

  • Prefer is in German?lang=de
  • Prefer is in Dutch?lang=nl
  • Prefer is in Italian?lang=it

Example:   ...stat/config/setup.php?lang=de

The PHP Web Stat will be shipped with german and english language support only.
Support for other languages are available on our website. Please unzip the downloaded language module and copy it into the language folder on your server.

Which language modules are available you can see in the language folder.
